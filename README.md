# Rami Score Counter

A modern web-based score counter for the Rami card game. This application allows two players to track their scores with various point increments (+1, +10, +20, +30) and decrement (-1, -10).

## Features

- Real-time score tracking for two players
- Support for various point values
- Responsive design for desktop and mobile
- Modern, card game-themed UI
- Current player turn indicator
- Reset game functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome for icons
- Google Fonts for typography
